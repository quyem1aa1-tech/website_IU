# 🌐 Website IU - Mạng Nội Bộ

Dự án mini-project xây dựng hệ thống website mạng nội bộ dành cho sinh viên/giảng viên IU.

## 🚀 Công nghệ sử dụng
* **Backend:** Java 25, Spring Boot 4.0.5
* **Database:** SQLite (Dễ dàng di chuyển, không cần cài đặt server DB phức tạp)
* **Build tool:** Maven

## 📂 Cấu trúc dự án
* `backend/`: Chứa mã nguồn Spring Boot.
* `frontend/`: Giao diện người dùng.
* `database/`: Chứa các script SQL và file `.db`.

## 🛠 Hướng dẫn chạy dự án
1. Clone repository: `git clone https://github.com/quyen1aa1-tech/website_IU.git`
2. Mở folder `backend` bằng IntelliJ hoặc VS Code.
3. Chạy lệnh Maven: `mvn clean install`
4. Run ứng dụng qua file `DemoApplication.java`.
